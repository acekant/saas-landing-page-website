## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/acekant/saas-landing-page-website.git
   ```
2. Navigate to the project directory:
   ```sh
   cd first_complete_project
   ```
3. Install dependencies for the backend:
   ```sh
   npm install
   ```

4. Start tailwind :
   ```sh
   npm run watch
   ```
